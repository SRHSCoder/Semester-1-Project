Web Development

The web is developed by scripting lanuages that tell the server what to display on the website.

HTML

Front-end Lanuages

Hypertext Markup Lanuage is the scripting lanuage that uses specific tags to create a website. HTML lastest version is called HTML5 and most web developers code with HTML5.

CSS

Cascading style sheet(s) is used to make website look better and easyier to look at. CSS latest version is called "CSS3" and is used by most developers. CSS uses the classes made in HTML and can add styles to them, such as the font size, or the height of the navigation bar at the top of the page. CSS3 also has "displays" that can make the site more functional such as Flex, Grid and more.

JavaScript

JavaScript is a programming lanuage that makes the website far more fuctional. If there is an error on the website, JavaScript can tell and tell the user (client) that there is an error. JavaScript is a Front-end Language and can be used as a Backend lanuage.
