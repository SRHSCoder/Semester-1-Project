Protocols

HTTP

Hypertext Transfer Protocol is a protocol for transfering website across the web.

HTTPS

HTTPS stands for Hypertext Transfer Protocol over Secure Socket Layer. It is a secure version of HTTP.

FTP

File transfer protocol. Used for when transfering or sharing files over via internet or website. Some componies sell software through this protocol.
